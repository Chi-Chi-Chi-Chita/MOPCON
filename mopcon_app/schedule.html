自從接觸到 IoT 後，身為一個硬體人的我開始覺得，沒有軟體思維真的是萬萬不行。我和 IoT 的緣分，最早是從學習 ZigBee 開始，而後我又發現光是會寫晶片韌體，真的是萬萬不行。後來，我遇到了 Node.js，試著用它來寫 ZigBee 驅動程式，在完成的那一天我又驚覺萬萬不行了。隨後，開啟了我學習 Web 前後端的道路。一頭栽進軟體領域，直到今天我還是像個嬰兒般，走得跌跌撞撞。我希望在這場演講介紹一點熱門的 IoT 通訊協定，再穿插一點我的學習感想，邀請大家在這 50 分鐘裡，和我一起夢遊仙境。

李健榮(Simen)

Tera - 無限容量、無限想像

Tera 是和沛團隊過去五年重要的研發成果。它使用雲端儲存服務 (Google Drive, SWIFT, AWS S3)，讓 Linux 電腦可以掛載 HCFS (Hopebay Cloud Filesystem)，使本地儲存空間看來跟雲端一樣大。本地儲存變成快取，讓存取效能不會受到網路的延遲。目前 Tera 可以用在 Android 4.4 - 7.1 的手機上。
